# Vibeland Portfolio

Это стартовая запись для мини‑раздела “проекты/блог” на Markdown.

## Что сделано

- Nuxt 4 + Tailwind v4
- Светлая/тёмная тема на CSS‑переменных
- Контактная форма, отправка на `/api/contacts`
- Серверное сохранение обращений в SQLite (`.data/vibeland.sqlite`)

## Стек

- Nuxt 4, Vue 3
- TypeScript
- Tailwind v4
- Nitro API
- SQLite
